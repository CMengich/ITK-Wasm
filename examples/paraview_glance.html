Use the Vue.js UI Framework with itk-wasm to create a Web App

ParaView Glance is a light-weight, open-source, web application developed at Kitware for visualizing volumetric images, molecular structures, geometric objects, and point clouds.

ParaView Glance is based on the Vue.js user interface
framework.

ParaView Glance uses itk-wasm to provide segmentation, registration, and analysis algorithms that can be applied to data loaded with the app. All processing occurs within your browser, and no software needs to be downloaded or installed on your machine.
