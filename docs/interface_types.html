Interface Types

itk-wasm execution pipelines support the following interface types:

  • TextFile
  • BinaryFile
  • TextStream
  • BinaryStream
  • Image
  • Mesh
  • PolyData

These interfaces types are supported in the Emscripten interface, WASI embedding interfaces, and native or virtual filesystem IO.

The following itk::wasm::Pipeline components can be included in a C++ to ingest and produce these interface types. For Input types, use Get() to get the corresponding C++ object value after ITK_WASM_PARSE_ARGS is called. For Output types, use Set(value) to output the value before main exits. For example,

InputTextStream
A string. To reader this data type in C++, using the resulting std::istream.
