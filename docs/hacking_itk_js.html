Build dependencies

  • Node.js / NPM
  • Docker
    • On Linux, make sure you can run docker without sudo.
    • On Windows, make sure Shared Drives are enabled in the Docker settings.
  • Bash
    • Bash is installed by default on Linux and macOS.
    • On Windows, we recommend Git Bash.
