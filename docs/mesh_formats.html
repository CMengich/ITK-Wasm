Supported Mesh and Point Set File Formats

Read and write to and from itk/Mesh is supported. For visualization with vtk.js, use the MeshToPolyData WebAssembly pipeline included with itk.js.

  • BYU Geometry File Format
  • FreeSurfer surface file formats, binary and ASCII
  • OFF Object Format
  • STL Stereolithography CAD software created by 3D Systems
  • Wavefront OBJ geometry file format
  • VTK legacy file format for vtkPolyData
