Supported Image File Formats

Read and write to and from Image is supported. For visualization with vtk.js, use convertItkToVtkImage.

  • BioRad
  • BMP
  • DICOM
  • DICOM Series
  • ITK HDF5
  • JPEG
  • GE4,GE5,GEAdw
  • Gipl (Guys Image Processing Lab)
  • LSM
  • MetaImage
  • MINC 2.0
  • MGH
  • MRC
  • NifTi
  • NRRD
  • Portable Network Graphics (PNG)
  • Tagged Image File Format (TIFF)
  • VTK legacy file format for images
  • Scanco microCT volume file format
  • Varian FDF
